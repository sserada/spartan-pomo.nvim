# Security Policy

## Reporting Issues

If you discover a security issue in spartan-pomo.nvim, please report it by:

1. Opening a GitHub issue with the `security` label
2. Or contacting the maintainer directly

Please include:
- Description of the issue
- Steps to reproduce
- Affected versions
- Any potential fixes you've identified

## Scope

This plugin runs locally in your Neovim instance and does not:
- Send data to external servers
- Access sensitive files beyond standard Neovim permissions
- Execute arbitrary code from external sources

## Updates

Security fixes will be released as patch versions and documented in the CHANGELOG.
